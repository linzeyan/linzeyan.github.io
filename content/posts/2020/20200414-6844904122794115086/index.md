- [Surprisingly, you can view images in the Linux CLI?](https://juejin.cn/post/6844904122794115086)

#### FIM

> `sudo apt-get install fim`

Common shortcuts for images in FIM:

- PageUp / Down: previous/next image
- +/-: zoom in/out
- a: auto scale
- w: fit width
- h: fit height
- j / k: pan down/up
- f / m: flip/mirror
- r / R: rotate (clockwise/counterclockwise)
- ESC / q: quit
