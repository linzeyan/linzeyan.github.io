Double slashes (//) - To specify full path of a file, you must use double slashes. For example, let us say you are editing a file named info.txt that is located in /home/sk/Documents/ directory of your remote system. In this case, the command would be: vim scp://sk@192.168.225.22//home/sk/Documents/info.txt
